## About the book

*Built for Humans* is a field guide for founders building AI-native products and applying to Y Combinator. Its core argument: YC partners aren't evaluating technical sophistication, they're evaluating obsession with a specific human problem and that obsession only shows up through real contact with the people who have it. The book moves through five parts: the mindset shift from building a "portfolio piece" to building for a user; how to find, validate, and distill a problem worth solving; how to use a multi-agent AI workforce to build and ship fast on a near-zero budget; how to write the YC application and handle the interview; and how to sustain that same user-obsessed discipline long after the application is submitted. It closes with practical appendices and a six-week execution checklist, a prompt library for a multi-agent build, and reference tables for the application itself.
